# Проект: Место
#### Описание:
 Место, где можно делиться фотографиями из своих путешествий. 
#### Что может пользователь:
+ Редактировать имя и описание. Менять аватор пользователя
+ Добавлять новые фото
+ Удалять свои фото
+ Ставить лайки
#### Список технологий: HTML, CSS, JavaScript
